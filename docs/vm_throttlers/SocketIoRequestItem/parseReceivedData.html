void parseReceivedData(receivedData)

Parses and handles any data received from the socket on a continuing basis.

Certain throttlers may filter the data that get's sent here. Notice that the receivedData parameter is types dynamic. Different sockets return different types of data, therefore it's up to the specific SocketRequestItem to handle it appropriately.
